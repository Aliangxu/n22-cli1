# n22-cli1

# 安装

全局安装 cli
npm install -g @n22/cli

# 使用模版初始化项目

n22 create <你的项目名称> <你需要的模版>

## 注：

- `你需要的模版`参数不写默认为`simple`模版
- 新的模板需要挂载在 git
  ```
  http://git.n22.com.cn:8089/rad/client-xxxxx-template.git
  其中xxxxx为你的模板名称
  ```

# 目前 git 存在的模板

- 1、`client-app-simple-template`移动端
- 2、`client-simple-template`PC 管理平台
  注：模板需要开通账号权限

# 进入项目

cd 你的项目名称

# 安装依赖

npm install

# 运行

npm run dev
